## 📱 Sobre o Projeto

**Nóbel** é um protótipo funcional de aplicativo de compras mobile, desenvolvido inteiramente em HTML, CSS e JavaScript puros — sem frameworks externos — e otimizado para telas de smartphone (390 × 844 px).

O projeto simula uma experiência de e-commerce completa, cobrindo os principais fluxos de navegação que um usuário encontraria em um app real:

- **Tela inicial** com header personalizado, barra de busca com sugestões em tempo real e destaques promocionais
- **Catálogo de produtos** organizado por categorias (Eletrônicos, Moda, Casa, Esporte, entre outras), com filtros de subcategoria e ordenação por preço ou relevância
- **Filtro de preço** por faixa personalizada ou presets rápidos
- **Tela de produto** com galeria de imagens, avaliações, seleção de variantes e botão de adicionar ao carrinho
- **Carrinho de compras** com controle de quantidade, cálculo automático de subtotal e simulação de checkout
- **Tela de favoritos** para salvar itens de interesse
- **Perfil do usuário** com histórico de pedidos e dados cadastrais
- **Barra de navegação inferior** com indicadores de aba ativa e badge de notificação no carrinho

---

## 🎯 Objetivos de IHC

O trabalho aplica conceitos centrais da disciplina:

- **Usabilidade** — navegação intuitiva, hierarquia visual clara e feedback imediato para cada ação do usuário
- **Design centrado no usuário** — fluxos construídos a partir de tarefas reais de compra online
- **Acessibilidade** — contrastes de cor adequados, áreas de toque generosas e tipografia legível
- **Affordance** — elementos visuais (ícones, botões, chips) que comunicam sua função sem necessidade de instrução
- **Consistência** — paleta de cores, espaçamentos e componentes padronizados em todas as telas
